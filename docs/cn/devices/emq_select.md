# 下拉选择

### 获取所有类型设备
> GET /emq_select/devices
##### Example response
```json
[
    {
        "attr": {
            "cloudProtocol": 3,
            "deviceIntID": 13,
            "deviceType": 1,
            "gatewayProtocol": null
        },
        "label": "end devices",
        "value": "123123124124124"
    },
    {
        "attr": {
            "cloudProtocol": 7,
            "deviceIntID": 10,
            "deviceType": 2,
            "gatewayProtocol": 7
        },
        "label": "gatways",
        "value": "2441aa371f1030d22ec18e8f1d1ff123"
    }
 ]
```

    {
        "attr": {

### 获取分组未加入的设备
> GET /emq_select/groups/<:id>/not_joined_devices
##### Example response
```json
[
    {
        "label": "device2",
        "value": 2
    }
]
```

### 获取证书列表
> GET /emq_select/certs
##### Example response
```json
[
    {
        "label": "cert",
        "value": 1
    }
]
```

### 获取证书未加入的设备
> GET /emq_select/certs/<:id>/not_joined_devices
* 只返回认证类型(authType)为证书(2)的设备
##### Example response
```json
[
    {
        "label": "cert device",
        "value": 6
    }
]
```
